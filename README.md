# Mere Sai
Сериал про Саи Бабу из Ширди.

Серии с русскими субтитрами есть здесь:

https://teleseriali.ru/564/

Сериал на Youtube:

https://youtu.be/Z-sXNXA7LJI?si=3wT3UTy88ALPSuk_

Многие серии на Youtube имеют английские субтитры, которые можно автопереводчиком перевести на русский (в настройках субтитров на Youtube). Но некоторые серии не имеют субтитров (например, серия 147: https://www.youtube.com/watch?v=fm2GPF1qzX8). 

Для серий, у которых нет субтитров на Youtube, используйте субтитры из .srt файлов в этом репозитории. Чтобы подключить внешние субтитры к видео на Youtube, можно использовать браузер Chrome и плагин к нему YouTube Subtitle Loader: 

https://chromewebstore.google.com/detail/youtube-subtitle-loader/lkffaclmmddbdaoccdndflefbjhpollp?hl=en
